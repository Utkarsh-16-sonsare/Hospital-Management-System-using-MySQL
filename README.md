A database-driven Hospital Management System designed to efficiently manage patient records, doctor details, appointments, and hospital operations using MySQL.

## Project Overview
This project focuses on designing and implementing a relational database system to streamline hospital workflows and ensure organized data management.

## Features
- Patient registration and record management  
- Doctor and staff information management  
- Appointment scheduling system  
- Structured relational database design  
- Optimized SQL queries for fast data retrieval  
- Data integrity through normalization and constraints

  <img width="1285" height="724" alt="image" src="https://github.com/user-attachments/assets/5d368a91-04c2-4cce-9497-3f6f141fbb5c" />


## Tech Stack
- **Database:** MySQL  
- **Language:** SQL  
- **Tools:** MySQL Workbench / phpMyAdmin  

## Database Design
- Entity-Relationship (ER) modeling  
- Normalized relational schema  
- Tables for Patients, Doctors, Staff, Appointments, Billing, etc.  
- Primary & Foreign key relationships
